Pointer :
---------
=> A pointer in C is a variable that store the memory address of another variable.

- pointers denoted by, * and it uses *ptr ,

- the datatype specifier for the pointer is %c

example:
#include<stdio.h>
int main(){
  int a=10;
  int *ptr;
  ptr=&a;

  printf("the Address: %c and the Value: %d", ptr,*ptr);
  *ptr=100;
  printf("\n a=%d",a);
return 0;
}
